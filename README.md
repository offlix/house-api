# House Api

## installation

## typescript installation

```
npm install -g typescript
```

## install packages

```
npm i
```

## run

```
npm start
```

# Endpoints

## Routes

```
/api/
```

### get health `(GET)`

```
/api/health
```

## House Routes `/api/house`

- ### create a house `(POST)`

```
/add : create a record for a house
```

- ### get houses `(GET)`

```
/ : get list houses with price
```

- ### get house by Id `(GET)`

```
/:id
```

- ### edit and update house `(PUT)`

```
/:id
```

- ### delete a house by Id `(DELETE)`

```
/:id
```
